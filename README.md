# Claude Dashboard

Real-time visualization dashboard for Claude Code JSONL logs

## Getting Started

```bash
npm install && npm start
```

## Features

- **Live Updates**: Real-time monitoring of Claude Code conversation logs
- **DiceBear Avatars**: Unique avatar generation for participants
- **Timeline with Markers**: Visual timeline showing conversation flow and key events
- **Haiku Narrator Mode**: Poetic summaries of conversation events

## About

This dashboard provides an interactive interface for visualizing and analyzing Claude Code JSONL log files. It offers real-time updates as conversations progress, making it easy to track and understand complex coding sessions.
